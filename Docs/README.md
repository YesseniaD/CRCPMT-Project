# Carpeta para alojar todos los documentos del proyecto de minería de textos.

Los documentos aquí contenidos, poseen información relevante del proyecto en función de las metodologías utilizadas para el desarrollo del mismo. 
Esta información se encuentra agrupada en tres categorías:

Categoría 1: /Project
1. Comprensión del negocio
a) Objetivos
b) Evaluación de la situación
c) Determinar los objetivos de la minería de datos

Categoría 2: /DataReport
2. Comprensión de los datos
a) Proceso de recolección de los datos iniciales
b) Descripción de los datos (diccionarios)
c) Proceso de exploración de los datos
d) Verificación de la calidad de los datos
 	- Proceso de selección de datos
 	- Proceso de limpieza de datos
 	- Proceso de construcción de datos
 	- Proceso de integración de datos
 	- Proceso de formateo de datos

Categoría 3: /Model
3. Documentos de planificación y ejecución del proyecto
a) Planificación de los experimentos
a) Reportes técnicos de la ejecución de cada versión y fase experimental
b) Información obtenida de los asesores del proyecto
c) Reporte final
